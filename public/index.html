ABOUT PAGE

Homepage Description

This is the home screen through which you can enter the application through which you can see a changing photo gallery and the names of the creators of the photos.

Navigation Menu

This is a navigation menu through which you can register for the application, connect to the application and read about it.

Sign-Up Menu

The Sign-Up Menu is where the user fills in details in the appropriate fields to register for using the application.

Login Menu

The login menu is the menu through which the user connects to the application. After logging in, the user is transferred to the home page and in the navigation menu there will be an option to disconnect from the application instead of the option to connect to it.

Add Picture Menu

In the Add Picture Menu, the user may add a picture that he owns, specify who created the picture , describe it with a title and description and give it a price tag.
